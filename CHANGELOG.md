# Changelog

本文件记录项目的所有重大变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] - 2025-02-05

### Added

- **通知系统**: 支持价格预警，可设置目标价格和方向（高于/低于）
- **设置对话框**: 支持刷新间隔、主题切换、自动刷新、显示盈亏等配置
- **全局错误处理**: ErrorHandler 类支持错误记录、统计、装饰器
- **加载状态优化**: FundCard 添加 ProgressRing 覆盖层，刷新按钮自动禁用
- **空状态处理**: 基金/商品/新闻列表为空时显示友好提示
- **数据导出功能**: CSV 导出基金列表和持仓报告
- **数据缓存层**: JSON 持久化缓存，支持 TTL 过期、统计和清理
- **集成测试**: 26 个测试用例覆盖完整工作流

### Changed

- 升级到 Flet 0.80.5 最佳实践
- 使用 `TabBar` + `Tabs` 组合替代旧版 Tab API
- 代码风格统一（ruff 自动修复 428 个问题）

### Fixed

- 错误处理优化，防止应用崩溃
- 加载状态反馈更及时
- 空状态显示更友好

### Technical

- 添加 mypy 类型检查配置
- 添加 ruff 代码风格检查配置
- 新增 120+ 个测试用例
