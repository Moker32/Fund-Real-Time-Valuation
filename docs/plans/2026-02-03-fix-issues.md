## 布局修改

### 修改前
- 使用 Grid 布局 `grid-columns: 2fr 1fr 1fr`
- 包含行业板块功能
- 三列显示较分散

### 修改后
- 使用 Grid 布局 `grid-columns: 1fr 1fr 1fr`（三等分）
- 移除行业板块（根据设计图不需要）
- 三栏用边框分隔，更紧凑
