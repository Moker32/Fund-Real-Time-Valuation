# Documentation

项目文档目录。

## 目录结构

```
docs/
├── API.md                      # API 接口文档
├── data-source-performance-report.md  # 数据源性能报告
├── data-sources-index-research.md     # 数据源指数研究
├── performance-evaluation.md   # 性能评估
├── api/                        # API 参考 (自动生成)
│   └── src/datasources/        # 数据源 API 文档
├── architecture/               # 架构设计文档
│   └── README.md
├── guides/                     # 教程和指南
│   └── README.md
└── plans/                      # 功能设计文档
    ├── README.md
    └── ARCHIVE/                # 已归档的设计文档
```

## 内容说明

| 目录 | 描述 |
|------|------|
| `API.md` | API 接口使用文档 |
| `api/` | 自动生成的 API 参考文档 |
| `architecture/` | 架构设计和技术决策 |
| `guides/` | 实用教程和操作指南 |
| `plans/` | 功能设计文档和待办事项 |
| `plans/ARCHIVE/` | 已完成或已废弃的设计文档 |
